О приложении

Инсулин помощник

Это статический сайт без backend и базы данных. Все настройки и история хранятся локально в браузере пользователя.

Приложение не заменяет врача и не назначает дозу инсулина. Оно предназначено только для прозрачного математического расчёта по данным пользователя.

Формулы расчёта

Расчёт по продукту

углеводы = граммовка продукта × углеводы на 100 г / 100 ХЕ = углеводы / грамм углеводов в 1 ХЕ точный расчёт = ХЕ × коэффициент инсулина на 1 ХЕ

Расчёт по ХЕ

точный расчёт = ХЕ × коэффициент инсулина на 1 ХЕ

Округление

нижний и верхний варианты рассчитываются по выбранному шагу дозы
